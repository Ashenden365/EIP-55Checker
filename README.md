# EIP-55 Checksum Checker

EVM 系ウォレットアドレスが **EIP-55 チェックサム表記として正しいかどうか** を判定する Streamlit アプリです。

## このアプリが確認するもの

このアプリは、入力された文字列について次の2点を確認します。

1. **20バイトの EVM アドレスとして形式が正しいか**
2. **EIP-55 に従った大文字・小文字の表記になっているか**

重要なのは、**EIP-55 はアドレスの「存在判定」ではない**という点です。  
EIP-55 は、アドレスの文字列表現にチェックサムを埋め込むためのルールです。

したがって、

- EIP-55 として不正
- しかしアドレス値そのものは 20 バイトの hex として有効

ということは普通にありえます。

## できること

- EVM アドレスの形式チェック
- EIP-55 チェックサム判定
- Keccak-256 ハッシュの表示
- 正しいチェックサム表記の表示
- 入力値と正しい表記の差分表示
- どの英字を大文字にすべきかの可視化
